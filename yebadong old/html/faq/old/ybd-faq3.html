About h.r.a.(han.rec.artrock)

han.rec.artrock 뉴스그룹은 현재 운영되고 있는 저희 예바동과 상호 연결되어 있는 han 뉴스그룹입니다. 예바동에 메일을 보내시는 경우 h.r.a.그룹으로 자동 forwarding이 되게 현재 설치되어 있습니다.

따라서, YBD mailinglist에 가입하지 않으신 분들도 자유롭게 h.r.a.뉴스그룹을 통하여 저희 메일링리스트에서 주고받는 정보를 접하실 수 있습니다.

About a.m.p.(alt.music.progressive)

a.m.p(alt.music.progressive) 뉴스그룹은 h.r.a.와 마찬가지로 USENET을 통하여 프로록/아트록에 관한 여러가지 정보를 주고받는 뉴스그룹입니다. 이 뉴스그룹은 Internet에서 접할수 있는 프로록 관련 정보중 가장 많은 양의 정보를 주고받는 매우 큰 뉴스그룹입니다. 정보의 양은 하루 평균 200개 정도의 article이 올라오는 정도로 매우 많습니다.

a.m.p.에 관한 더 자세한 정보를 원하시는 분은 a.m.p. FAQs을 찾아 읽어보시기 바랍니다.

Zeuhl Mailinglist

Magma/Zeuhl/RIO/New Music 에 관한 정보와 이야기를 주고 받을 메일링 리스트가 만들어졌습니다. 이름은 zeuhl 이고요, 지금은 처음 단계라서 가입 인원도 몇명 되지 않는 시험 단계이지만 곧 전세계의 Magma/Zeuhl/RIO/New Music 팬들이 이 메일링 리스트를 통해 만남의 기회를 얻게 되리라 생각합니다.

가입 메일을 보내실 주소는, majordomo@rivendel.lin.ufrj.br이고, 가입 명령어는 메일 몸체에 "subscribe zeuhl" 이라고 쓰시면 됩니다. 모든 명령어는 "Subject 란에다 쓰지 마시고 반드시 메일 몸체에 쓰셔야 합니다." "help" 명령을 보내시면 리스트 명령어에 관한 정보 메일을 받아보실 수 있습니다. 글을 올리실 때는 zeuhl@rivendel.lin.ufrj.br 로 보내시면 됩니다.

관심 있으신 분들의 적극적인 참여를 바랍니다!!

Internet Prog-rock Related Articles

인터넷에 Gibraltar Encyclopedia외에 또 하나의 방대한 프로그레시브 자료모음이 생겼습니다. 내용은 Peter Hammill/ VDGG 의 것이고 포스트 스크립트 파일로 총 165페이지의 책입니다. 포맷은 LaTeX이고 양면복사를 고려해서 만들어 졌습니다. 제네시스 연구팀의 가사모음집보다 훨씬 좋습니다. 내용을 좀 살펴보면,
1. Fables and Figments
2. Poems
3. Lyrics
의 세파트로 나누어져 있고 A4지 각 장마다 빽빽이 채워져 있습니다. 또한 가사파트에는 피터 해밀이 곡의 주인공 혹은 느낌을 직접 스케치한 그림도 있고 아주 좋습니다.

관련된 파일 및 Web Site는 다음과 같습니다.

  • ftp.informatik.rwth-aachen.de:/pub/reports/others/Hammill.a4.ps.gz
  • ftp.informatik.rwth-aachen.de:/pub/reports/others/Hammill.letter.ps.gz
  • http://www-i2.informatik.rwth-aachen/ghogen/personal.html

Mailing List

Mailing list란 본 YBD 처럼 email로 연결된 동호회 집단입니다. 대부분의 분들은 Mailing List의 정체에 대해서 잘 아시겠지만 이 YBD 글모음을 보시는 YBD 이외의 net을 잘 모르시는 다른분들을 위해 설명을 잠깐 드리자면...

예를들어 본 YBD의 경우 가입을 하시려면 administration 주소인 majordomo@yebadong.kaist.ac.kr로 email을 보내시면 가입이 되게 되어 있읍니다. 메일의 내용에는 subscribe yebadong이라고 꼭 쓰셔야 됩니다. 제목에 써서는 소용이 없습니다. 가입을 하신 이후에는 yebadong@yebadong.kaist.ac.kr로 날아오는 YBD member 들의 모든 mail이 한 copy 씩 배달되게 되어있고 같은 주소로 mail을 보내시면 YBD 전 member들에게 당신의 message가 echoe-out 됩니다.

예바동 Mailing List는 digest mode를 지원합니다. 예바동에는 비교적 많은 메일이 쌓입니다. 예바동 Mailing List의 Subscriber이면서도 자신의 mail box에 많은 메일이 쌓이는게 싫으신분은 digest mode에 신청하시면 됩니다. digest mode는 일주일에 두세번 꼴로 여러 메일을 하나로 묶어서 보내줍니다. 신청 방법은 majordomo@yebadong.kaist.ac.kr로 메일의 내용에 subscribe yebadong-digest라고 써서 보내시면 됩니다.

World Wide Web

속도가 느리긴 하지만 World Wide Web은 그야말로 거미줄 처럼 연결이 되어있어 YBD의 main prog home page에서 다른 group의 home page 로의 연결도 mouse click 한번으로 해결된다는 점, 그림 file 이 동시에 지원되므로 훨씬 더 시각적인 효과를 제공한다는 점, 등등의 장점으로 progressive 음악 관련 internet 매체로서도 최근 굉장한 각광을 받고 있읍니다.

FTP site

FTP site에서는 각종 Mailing list들의 back issue들 progressive rock group 의 앨범 jacket 사진, 공연모습, 심지어는 공연 ticket등의 그림 file들과 실제로 computer에서 play 하여 들을 수 있는 Progressive rock음악 sampler file 등을 얻을 수 있읍니다.

다음은 각종 progressive rock 관련 file들이 store 되어있는 ftp site들입니다.

    ortega.cs.ucdavis.edu : 이 site에는 각종 articles, concert reviews, discographies, interviews, Gibraltar backissues, album reviews, 그리고 a.m.p.의 최신 FAQ files가 있습니다. Administrator는 Mike Borella 이고 그의 email 주소는 borella@cs.ucdavis.edu 입니다. 자세한 궁금증은 그에게 문의하시기 바랍니다.

    ftp.uwp.edu : 이 site는 각종 메일링리스트의 archive들을 비롯한 여러 정보를 갖고 있습니다.

    techno.stanford.edu : It is a repository of information about all types of ambient music, from Eno's `pure' form to ambient house to droning guitar/psych rock. It also has a gopher connection as well. If you're not sure you can ftp ask your sysamdin folks or if you're on a unix system type `man ftp' to see if you have an online man page. Emacs has a marvellous facility for doing transparent ftp called `ange-ftp'. Check out the information using emacs' `info' facility (C-h i usually). You can usually ftp from a web browser too.
